🔊 Test Chức Năng Âm Thanh Extension

Nhấn các nút bên dưới để test chức năng âm thanh:

Hướng dẫn:

  1. Mở Chrome Developer Tools (F12)
  2. Nhấn các nút test ở trên
  3. Kiểm tra Console để xem log
  4. Nghe âm thanh phát ra

Lưu ý:

  • Chrome TTS cần extension context để hoạt động
  • Web Speech API hoạt động trong trang web
  • Extension sẽ sử dụng Chrome TTS từ background script
